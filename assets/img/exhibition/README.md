# Exhibition Booth Images

Place your booth preview images in this folder using the naming convention below.

## Naming Convention

Use lowercase package names and size codes, separated by hyphens:
`{package}-{size}.jpg` or `{package}-{size}.png`

## Required Images

### Basic Package
- `basic-3x3.jpg` or `basic-3x3.png` - (3mx3m) booth
- `basic-6x3.jpg` or `basic-6x3.png` - (6mx3m) booth

### Silver Package
- `silver-3x3-corner.jpg` or `silver-3x3-corner.png` - (3mx3m) corner booth
- `silver-3x3-ushape.jpg` or `silver-3x3-ushape.png` - (3mx3m) u-shape booth
- `silver-6x3-corner.jpg` or `silver-6x3-corner.png` - (6mx3m) corner booth
- `silver-6x3-ushape.jpg` or `silver-6x3-ushape.png` - (6mx3m) u-shape booth

### Gold Package

- `gold-6x3-corner.jpg` or `gold-6x3-corner.png` - (6mx3m) corner booth

## Notes
- Images will auto-load when named correctly
- The system tries .jpg first, then .png
- Recommended dimensions: 640x440px or similar 16:11 ratio
- If no image exists, the preview area hides gracefully
